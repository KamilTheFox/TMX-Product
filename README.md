# Продукт для ТМХ

## Приложение сбора дефектов вагонов метро

Проект продолжение ТМХ-Хакатона (Репозиторий тоже имеется)

Реализация включает в себя большое количество поправок, включая отказ от геймплея в мобильном приложении в угоду безопасности от преднамеренного вандализма от стимула геймификации

1. Минималистичный дизайн UI/UX 
2. Убраны все поля ввода данных для пользователя
3. Реализована панель модерации и администрации
4. Минималистичный дизайн архитектуры
5. Система банов аккаунтов.
